What I Built

A complete PDF-to-JSON extraction pipeline for NCERT Class 11 & 12 textbooks, designed to produce structured training data for LLMs.

How It Works

1. PDF Ingestion: PDFs are uploaded via FastAPI REST endpoints or processed in batch from a directory.

2. Page Rendering: Each page is converted to high-res PNG using PyMuPDF (fitz) for downstream processing.

3. Text Extraction: Two strategies available — PyMuPDF for clean text-based PDFs, and Tesseract OCR for scanned pages.

4. Content Structuring: Physics-specific structurer parses raw text into sections, examples, exercises, in-text questions, tables, and LaTeX equations using regex patterns.

5. Image Processing: OpenCV removes NCERT watermarks from figures; figures are extracted and linked to captions.

6. Output: Clean JSON with book metadata, structured content, and statistics — ready for PostgreSQL storage or direct LLM training.

Tech Stack

Backend: Python, FastAPI, Pydantic | PDF: PyMuPDF, Tesseract, Poppler | AI: Docling ML | Images: OpenCV | DB: PostgreSQL with JSONB
